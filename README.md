# LottoAnalysis
(玩具)-台灣彩券基於歷史開獎紀錄的開獎號碼預測

這是為了滿足家父願望的一個玩具程式。
基於台灣彩券的歷史開獎紀錄，透過計算每一個號碼在歷史開獎的頻率、週期、次數、去計算出下一期可能出現的開獎號碼

由上至下的排序是舊>新(sort is old to new)
